# 🚀 Meu Portfólio Pessoal

Bem-vindo ao meu repositório de portfólio! Este projeto foi desenvolvido para apresentar minhas habilidades como **Desenvolvedor Front-end**, focando em uma interface moderna, responsiva e com animações fluidas.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica.
- **Tailwind CSS**: Estilização rápida e responsiva via CDN.
- **JavaScript**: Lógica para interações e envio de formulário.
- **Font Awesome**: Ícones sociais e de interface.
- **Google Fonts (Inter)**: Tipografia moderna.

## ✨ Funcionalidades

- **Efeito Typing**: Animação de escrita no nome principal.
- **Animações de Entrada**: Ícones sociais que "sobem" para a posição final ao carregar.
- **Design Responsivo**: Adaptado para dispositivos móveis e desktop.
- **Carrossel Estilo Netflix**: Exibição dos projetos de forma dinâmica.
- **Formulário de Contato**: Integrado para recebimento de mensagens.
